# Java2048

***Combine the numbers until you reach 2048 or more!***

An implementation of the classic **2048** game, built with Java 21 and JavaFX, for learning purposes.



## Gameplay preview

![Image](https://github.com/user-attachments/assets/062c1a51-4fc8-4ffe-9251-66818e105633)



## Features

* Classic 2048 gameplay
* Smooth tile merging
* Score tracking
* JavaFX UI



## Built With

* Java 21
* JavaFX 21
* Maven
* jlink
* jpackage



## How to Run

1. Go to the **Releases** section.
2. Download the latest version (`v1.0.0` or newer).
3. Extract the ZIP file.
4. Open the `Java2048` folder.
5. Double-click: Java2048.exe
6. Windows may display a security warning on first launch.
   * To run the application:
    1. Click More info
    2. Click Run anyway

No Java installation required.
